# ConvertKit Slate Plugins

A collection of plugins for SlateJS maintained by the
[ConvertKit](https://convertkit.com/) team.

## Plugins

### [slate-code](/plugins/slate-code)

Adds code editing behaviors to a SlateJS editor.

### [slate-keymap](/plugins/slate-keymap)

A plugin for simplifying keyboard shortcuts in a Slate editor.

### [slate-lists](/plugins/slate-lists)

Adds list editing behaviors to a SlateJS editor.

## Testing

### [slate-testing-library](/slate-testing-library)

**Note: This library is experimental and may change**

A library to help with writing tests for SlateJS editors.
