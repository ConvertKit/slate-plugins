# Slate Lists

![](https://img.shields.io/circleci/project/github/ConvertKit/slate-plugins/master.svg?style=flat)

[View on Github](https://github.com/ConvertKit/slate-plugins)

A plugin for lists in SlateJS.

**Keybindings**

- `Enter`: insert new list item
- `Tab`: increase the depth of an item
- `Shift+Tab`: decrease the depth of an item
- `Delete` or `Backspace` at the start of an item: remove the item

## Install

```bash
yarn add @convertkit/slate-lists
```

## Usage

```jsx
...
import { Editor } from "slate-react";
import Lists from "@convertkit/slate-lists"

const plugins = [Lists({
  blocks: {
    ordered_list: "ordered-list",
    unordered_list: "unordered-list",
    list_item: "list-item",
  },
  classNames: {
    ordered_list: "ordered-list",
    unordered_list: "unordered-list",
    list_item: "list-item"
  }
})]

const MyEditor = (props) => (
  <Editor
    ...
    plugins={plugins}
  />
)
```

## Commands

### `wrapList({ type: String? })`

Wraps the current selection in a list.

### `unwrapList()`

Unwraps the list at the current selection.
